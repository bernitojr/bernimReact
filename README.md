# 🎮 BernimReact — Petit jeu React avec thème clair/sombre et confettis

Bienvenue dans **BernimReact**, une petite application React développée avec Vite.  
Elle inclut un thème clair/sombre, des animations de confettis, une interface responsive et une structure de composants propre et moderne.

jouable à cette adresse : 
[https://virgilebernard.github.io/bernimReact/](https://bernitojr.github.io/bernimReact/)
---

## ✨ Fonctionnalités

- 🎨 **Thème clair / sombre** avec bascule animée  
- 🎉 **Confettis** lors de certaines actions (canvas-confetti)  
- ⚛️ **Architecture React moderne** (hooks, composants modulaires)  
- ⚡ **Build ultra rapide** grâce à Vite  
- 📱 **Interface responsive**, testée sur mobile  
- 🚀 **Déploiement automatique** sur GitHub Pages

---

## 🛠️ Technologies utilisées

| Technologie | Rôle |
|------------|------|
| **React 19** | UI et logique |
| **Vite** | Dev server + build |
| **canvas-confetti** | Effets visuels |
| **ESLint** | Qualité du code |
| **GitHub Pages** | Hébergement |
